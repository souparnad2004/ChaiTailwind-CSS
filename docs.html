Installation

Initialize the engine by passing your config object to the constructor. 

import ChaiTailwind from "./chaiTailwind.js";
import config from "./style-config.js";

const chai = new ChaiTailwind(config);
chai.init();

Spacing (Padding & Margin)

Use prefixes like p, px, m, or mt followed by the scale value.

chai-p-2 (8px)
chai-p-8 (32px)
